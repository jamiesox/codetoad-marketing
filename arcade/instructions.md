# CodeToad Marketing Assets: Instructions

This document provides instructions for using the interactive demo and the Arcade recording script.

## Interactive Demo

The interactive demo is a self-contained HTML file that showcases the CodeToad user interface. It is located at `/home/ubuntu/codetoad-demo/index.html`.

### To run the demo locally:

1.  Navigate to the `/home/ubuntu/codetoad-demo` directory.
2.  Start a simple Python HTTP server: `python3 -m http.server 8080`
3.  Access the demo in your browser at `http://localhost:8080`.

### Demo Features:

*   **8 Screens:** The demo includes 8 key screens from the CodeToad application.
*   **Interactive Hotspots:** Green pulsing hotspots provide additional information about key features.
*   **Clickable Navigation:** Users can navigate through the demo using the top step navigation or the "Next Step" and "Previous" buttons.
*   **Rendered Components:** The landing, sign-in, and pricing pages are rendered with HTML/CSS for a more realistic experience.

## Arcade Recording

The Arcade recording script provides a shot-by-shot guide for creating a ~60-second product demo video. The script is located at `/home/ubuntu/codetoad-arcade/recording_script.md`.

### Recording Steps:

1.  **Set up your recording environment:** Use a screen recording tool like Loom or QuickTime.
2.  **Follow the script:** The script is divided into 6 scenes, each with specific visual cues, narration, and on-screen text.
3.  **Use the interactive demo:** The interactive demo can be used to capture the visuals for each scene.
4.  **Add narration and on-screen text:** Record the narration and add the on-screen text overlays in a video editor.

### Key Visuals to Capture:

*   **Landing Page:** Zoom into the headline.
*   **Dashboard:** Show repository import and health scores.
*   **Health Report:** Pan across the score ring and issue distribution.
*   **AI Workspace:** Show the file tree, code editor, and AI chat.
*   **AI Chat:** Show a user query and the AI's response.
*   **Pricing Page:** Show the different pricing plans.
